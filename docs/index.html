色分析 | Color Analysis

画像をアップロードすると、RGB / HSV / CIE L*a*b* の色空間マッピング、 K平均法による代表色抽出、マンセル表色系へのマッピングを行い、 分析画像一式(サマリーシートPDF/PNG含む)をZIPでダウンロードできます。
処理はすべてブラウザ内で実行され、画像が外部に送信されることはありません。

作品情報
起動中…

初回はライブラリ(数十MB)の読み込みに1〜2分かかります。2回目以降はブラウザキャッシュにより高速化されます。
RGB→マンセル変換は sRGB → CIE XYZ (D65) → 色順応変換(CAT02, D65→C光源) → xyY → Munsell Renotation Data (ASTM D1535) の補間、という手順で行っています。
